# computer_vision_js

Code examples with the usage of popular Computer vision libraries in Javascript

Libraries Used:
 
 - JsFeat
 - TrackingJs
 - HeadTrackr
 
 Dependencies installation:
 
 ```
 npm init 
 ```
 Server Initalization: 
 ```
 npm run serve
 ```
 Build Initalization: 
 
 ```
 npm run build
 ```
 Clean build:
 ```
 npm run clean
 ```
